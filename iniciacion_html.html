Añadir imágenes a le página web

Para hacer esto se utiliza el tag img, dentro de este se añaden atributos que especificarán detalles de la imágen.Estos atributos no necesitan de < > y tras ellos hay que poner el signo = y lo que se quiere poner entre "".

  • El atributo src de utiliza para poner la url de la imágen, su procedencia.
  • alt="" se utiliza para poner una pequeña descripción de la imágen en caso de que esta no se carge o de que una persona ciega visite la web.
  • width="" y heigth="" se usan para determinar la cantidad de píxeles, su tamaño.
