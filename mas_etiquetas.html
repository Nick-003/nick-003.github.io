Tablas

Se pueden crear tablas con las etiquetas:

  • <table>
  • Para en emcabezado: <thead>.
  • Para el cuerpo: <tbody>.
  • Para empezar a poner los elementos: <tr>.
  • Para añadir elementos dentro de head: <th>.
  • Para añadir elementos dentro de body: <tb>

Ejemplo:

Día Actividad Hora
Lunes Gimnasio 20:00
Martes Tenis 18:00
Miércoles Boxeo 19:00

Comentarios

Para realizar especificaciones sobre algún aspecto de la página se puede añadir texto que no es legible por el navegador, solo por el resto de personas que ven el código.
