Introducción a CSS

Para determinar características sobre partes de la página web se utiliza el lenguaje CSS. CSS son las siglas para Cascade Style Sistem, traducido como Sistema de estilo en casacada.

Métodos para incluir CSS

  • Con la etiqueta <style> dentro de <head>
  • En un fichero externo que se aplique a todo el documento de la página web mediante vínculos

Especificación de que se quiere seleccionar

Denpendiendo de si se quiere aplicar a todo el texto a a algo específico se utiliza:

  • Seleccionar por nombre

    • Se pone el tag por ejemplo p y a continuación { }, dentro de ellos se especificará qué característica se quiere definir.

  • Seleccionar por id

    • Se crea un id para especificar el elemento y se pone dentro de style con #
