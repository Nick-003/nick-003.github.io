Propiedades de texto en CSS

Se utilizan para cambiar el diseño y la presentación de la página web.

Font Family

Es la fuente de texto con la que se escribe, se utiliza el atributo font-family, puede ser de dos tipos.

  • Generic font family

    • Es un afuente genérica que es reconocible por todos los navegadores. Algunos ejemplos son:
    serif
    sans-serif
    monospace

  • Especific font family

    • Fuente específica que puede no estar presente en el navegador, se suele poner una genérica en caso de que no se reconozca.

Font size

Determina el tamaño con el que se presentan las letras, se usa el atributo font-size. Dos de las medidas más usadas son:

  • Font size px

    • Px son los píxeles en los que queremos que se muestre la imagen.

  • Font size em

    • Em es una medida relativa que marca la relación entre el texto por defecto con lo señalado con em

Font style

Se utiliza para poner cursiva a un párrafo completo sin abusar del tag em con el atributo font-style italic

Font weigth

Se usa para poner el texto en negrita sin abusar del tag strong copn el atributo font-wiegth bold

Text aling

Se utiliza para determinar la alineación del texto, que puede ser centre, rigth o left con el atributo text-align

Line height

Es la altura de una casilla remarcada por lineas.
Se puede especificar con valores como px o em.

Text decoration

Se usa para determinar si queremos decorar o no el texto,un ejemplo sería el subrayado utilizando underline o none dentro del atributo text-decoration.
